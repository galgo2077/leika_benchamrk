# Leika Benchmark Suite

Benchmark harness for Leika.

- Lives in its own repository under `~/Documents/leika_benchmark`
- Uses the installed `leika` package via `pip install leika`
- Keeps generated reports and results out of the engine library repo

## Run

```bash
./benchmark.sh
```

If needed, install or refresh the `leika` package first:

```bash
pip install leika
./benchmark.sh
```
